commands:
1. git init: hamare project ko github ke sath link karta hai...git hamari repository ke sath initialise ho gyi hai. Niche terminal m gitbash open karenge and git init chalayenge...isse .git file add ho jati hai.
2. ls -a: Now is file ko dekhna ho toh...niche wahi gitbash terminal m... ls -a..ye hidden file ko dekhne m help karti hai.
3. git status: ye dikhata/locate/track karta hai jo file mere github pe commit nai hui hai
4. ls: list directory...jitni file hongi wo ajayegi
5. git add . : ye sari files ko git pe dal dega...another way to add particular file... git add file1.txt...it will add this particular file
Ab agr git status karenge toh pta chalega...sari files ab tracked hai
6. git commit -m "v1" ..... -m is flag
Ab karenge git status toh show karege 
7. git log: Ye btayegi abi tak mere kitne commit hue hai...koi branch to nahi create kar rakhi hamne.
Har commit ki unique id hoti hai...
q to exit from log
8. git checkout id(jis id pe jana hai wo id dal denge) : agr purane wale version pe jana hai toh. Har commit ek unique id hai(git log karne pe dikh jayegi)...
9. git checkout d55f034fde17cf260d1384461f2e0e87b60d44f0 (ye id copy kar di)....isse
10. git checkout main(main ya master depend karta hai jha se commit ki hai)
git log se check kar lenge kya pehle jaisa ho gya hai
