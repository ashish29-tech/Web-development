What's web ?
Information system which is a collection of web pages, websites that we can access. Which can be shared with the help of internet.

Webpage: Static stuff. It's a single page with information...whose access is available everywhere on the internet.

Website: Collection of webpages(static stuff).

Webapplication: more interactive websites are Webapplications. Ex: Irctc vs Instagram.

Internet: Global network of interconnected devices. Which communicates via protocol TCP/IP.
TCP/IP: Transmission Control Protocol/Internet Protocol. When we interact with the internet, we need to follow some rules(made by organisation...) to interact with each other.

Global Network: connected by wires...spread under the ocean.

How does web works ?
It contains...client, server, DNS(Domain Name System), Data Centers
Client: Anyone who is requesting for the data.
Server: Understands your request and sends you the desirable output.
